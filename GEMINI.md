# Project Context: Jittirut Ong - Product Designer Portfolio

This project is a modern, minimalist portfolio for Jittirut Ong, a Product Designer from Thailand.

## Technical Stack
- **Frontend:** Vanilla HTML5, CSS3, and JavaScript.
- **Styling:** Custom Vanilla CSS with a focus on typography, spacing, and a modern light/dark mode theme.
- **Deployment:** Targetted for GitHub Pages (using the `docs/` directory).

## Architecture & Conventions
- **Directory Structure:**
  - `docs/`: Contains the production-ready site (HTML, CSS, JS).
  - `plan/`: Stores implementation details and project roadmap.
- **Design Philosophy:** Minimalist, high contrast, and accessible.
- **Theme:** Default is a light theme, with a manual toggle for dark mode.
